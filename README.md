# Semester Project 1

Semester Project 1 is a responsive science museum website developed during my frontend studies at Noroff. The project focused on creating an engaging and accessible experience for different target audiences while improving layout structure and responsive design skills.

## Features

- Responsive design
- Mobile and desktop layouts
- Interactive CTA buttons
- Hover effects and visual feedback
- Multiple informational sections
- Science and space themed design

## Technologies Used

- HTML
- CSS

## Improvements Made

- Improved CTA button styling
- Added smoother hover effects and visual feedback
- Updated button colors and shadows to better match the project theme

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Sander12312/SemesterProject.git
```

2. Open the project in VS Code.

3. Run the project using Live Server.
